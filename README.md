# RxSocketIO
Simple RxJava2 socket.io library wrapper over [socket.io-client-java](https://github.com/socketio/socket.io-client-java) library.

## Install

### Gradle

```gradle
repositories {
    jcenter()
}

dependencies {
  implementation 'com.gorgexec.mvvmcore:mvvmcore:1.0.7'
  annotationProcessor 'com.gorgexec.mvvmcore:compiler:1.0.7'
}
```
